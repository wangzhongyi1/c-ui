## 介绍

一个完整的 `UI组件库` 需要包含三部分：

1. 组件库代码

```text
包含编写组件库的源代码、esModule和umd双格式的打包文件
```

2. 示例代码

```text
一般放在项目根目录的 examples 文件夹下

是一个引入了组件库的完整示例项目
```

3. 组件库文档

```text
一般使用 vue-press、vite-press、storybook 等使用 Markdown 语法来帮助我们快速开发文档项目
```
